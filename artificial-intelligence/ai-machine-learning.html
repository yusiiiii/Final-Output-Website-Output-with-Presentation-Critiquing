User: What is machine learning in simple terms?
Machine learning is a field of study where algorithms learn patterns from data to make predictions or decisions without being explicitly programmed. It includes supervised, unsupervised, and reinforcement learning methods widely used in image recognition, language processing, and recommendation systems.
